{innks blog} v6 여섯번째 도전 블로그

이 블로그는 Hexo를 기반으로 hueman테마을 적용 했으며 개인적으로 필요한 부분을 수정도 했습니다.

소개

  • ID : innks
  • e-mail : algiddl11@gmail.com
  • Google+ : https://plus.google.com/+algiddl11

innks blog는 개인적인 공간의 블로그입니다. 때론 작업 공간이고, 저장 공간이며 노트 정리용블로그입니다.
파워블로그도 아니고 전업블로그도 아니며 그냥 블로그를 좋아하는 사람입니다.

애드센스…
블로그에 설치한 애드센스는 저에게 작은 선물과도 같은 것입니다. 보기 힘드시면 그냥 선물이라고 좋게 생각해 주시면 감사하겠습니다.5

이 블로그는 오타와 편견과 개인적인 글로만 있을수 있습니다. 다소 불편하시더라도 이 사람은 이렇구아 하며 작은 댓글 부탁 드립니다. :)

HIStory 2001~

  • 2001 : 군대 선임을 만나 인터넷이라는 것을 알게 되며 개인 홈페이지를 시작합니다. (rain4x.net)
  • 2003 : 몇년간 인터넷을 배우기 시작합니다. 웹마스터라는 과정을 거치고 블로그를 개설합니다. (3rd.pe.kr)
  • 2005 : 홈페이지를 만들어 주고 돈도 조금 만지기 시작하지만 오프라인에서 하는 일이 더 돈이 되는 관계로 블취미로 하기로 합니다.
  • 2007 : 드디어 디카 샀지만 출사는 가지 못하고 가끔 레이싱걸을 찍으러 다닙니다. 본격적으로 블로그 시작 합니다.
  • 2008 : 작은 회사 다니면서 블로그를 하지만 이때까지 홈페이지를 만들고 있습니다.
  • 2009 : 드디어 작은 독립. 모든것을 리셋하고 인터넷이라는 바다에 본격적으로 들어가 봅니다. 하지만 이미 IT쪽은 환갑인 상태… IT블로그 만.들었는데 약간의 SVN를 시작합니다.
  • 2010 : 서울의 작은 방에서 컴터 4대와 함깨 생활하기 시작합니다. 처음에는 이렇게 시작하는 것이지… 잡스 아저씨를 보고 작은 꿈을 키우기 시작합니다.
  • 2010.11 : 잠깐 다른곳에 갔다가 돌아오곤합니다. 하지만 2010년에 드디어 프로 블로그라는 것에 도전하기 시작합니다.(r5xs.textcube.com)
  • 2011.3 : 텍스트큐브중단으로 티스토리 이전.(r5xs.textcube.com -> r5xsv3.tistory.com/)
  • 2012 : 블로그 잠적 중단합니다.(r5xsv3.tistory.com) 이 길이 아닌것을 이제야 알기 시작합니다.ㅂ
  • 2013 : 좀더 넓은 곳을 보기 위해 여행을 다녀 봅니다. 디카와 함께 … 하지만 공허함이 많이 밀려 드는것 같네요.
  • 2014.3 : 아직 꿈을 버리지 못하고 블로그 다시 시작합니다. (the3.tistoey.com)으로 변경해서 시작.
  • 2014.5 : 이제부터 스킨 작업에 맛을 알았다 싶을 정도로 스킨 작업에 몰두 합니다. 이미지 위주의 블로그에서 텍스트 위주로 블로그가 만들어지고 글로만 해도 디자인이 되는것을 알게 됩니다.
  • 2016.12 : Git라는 곳으로 블로그 이적 계획을 잡고 옛날 서적들을 보기 시작합니다. Github의 정적블로그 있다는 글을 읽고 다시 불타 봅니다.
  • 2017.01.10 : http://innks.gitub.io 라는 곳으로 블로그 이적합니다.
