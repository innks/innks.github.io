MY SOFTWARE 소프트웨어

컴퓨터 소프트웨어(computer software, 혹은 간단히 소프트웨어, 문화어: 쏘프트웨어, 순화 용어: 무른모)는 저장장치에 저장된 특정한 목적의 하나 또는 다수의 컴퓨터 프로그램을 뜻한다. 프로그램 소프트웨어는 컴퓨터 하드웨어에 직접 명령어를 주거나 다른 소프트웨어에 입력을 제공함으로써, 그것이 수행하도록 구현된 기능을 수행한다.

“소프트웨어”라는 용어는 1957년에 존 터키(John W. Tukey)가 처음 사용한 용어이다. 일상적으로 이 용어는 응용 소프트웨어의 의미로 자주 쓰인다. 컴퓨터 과학과 컴퓨터 공학에서 “컴퓨터 소프트웨어”는 컴퓨터 시스템, 프로그램, 데이터에 의해 처리된 모든 정보를 말한다.

컴퓨터 소프트웨어는 컴퓨터 하드웨어의 반대 의미로, 컴퓨터 하드웨어는 소프트웨어가 실행되고 저장되는 물리 구조이다. –위키

“앗! 파일명이 기억이 안나 ㅠㅠ 어떻게 찾지?”

이런 문구를 읽고 앗! 이것이다. 제목으로 무얼 적을까 하다가 이 말이 제일 어울리지 않을까 싶을 정도네요.

  • 설명: windows 7 USB부팅 디스크 만들기 입니다. 물론 USB 8G 이상이고 PC가 USB부팅이 되어야 됩니다.

    https://www.microsoft.com/ko-kr/download/windows-usb-dvd-download-tool

# balabolka 바라볼카

  • 설명: Balabolka는 텍스트 음성 변환(Text-To-Speech, TTS) 프로그램입니다. 즉 Txt파일을 불러넣어서 컴퓨터가 읽어 주는 프로그램입니다.
  • 최신버전 다운로드: http://www.cross-plus-a.com/kr/balabolka.htm
    • 윈도우 XP/Vista/7/8/10 에서 동작합니다.
  • MF download : LinkPage -windows(US) v2.15.0.691
  • 한글 언어 패치는 영어 설치하고 프로그램 실행 후 한국어로 변경 하면 됩니다.
    • 한국어 TTS 앤진 : MSSpeech_TTS_ko-KR_Heami
    • 설치후 microsoft Speech Platform을 클릭 펼침 메뉴에서 한국어(ko-KR, Heami)로 선택 하면 됩니다.
    • 않될시(windows7 x64) : Microsoft Speech Platform - Runtime Languages (Version 11) 설치 -> heami_64.reg 실행 (reg는 않될때 하세요.)
