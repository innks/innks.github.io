MY SOFTWARE MY 소프트웨어

컴퓨터 소프트웨어(computer software, 혹은 간단히 소프트웨어, 문화어: 쏘프트웨어, 순화 용어: 무른모)는 저장장치에 저장된 특정한 목적의 하나 또는 다수의 컴퓨터 프로그램을 뜻한다. 프로그램 소프트웨어는 컴퓨터 하드웨어에 직접 명령어를 주거나 다른 소프트웨어에 입력을 제공함으로써, 그것이 수행하도록 구현된 기능을 수행한다.

“소프트웨어”라는 용어는 1957년에 존 터키(John W. Tukey)가 처음 사용한 용어이다. 일상적으로 이 용어는 응용 소프트웨어의 의미로 자주 쓰인다. 컴퓨터 과학과 컴퓨터 공학에서 “컴퓨터 소프트웨어”는 컴퓨터 시스템, 프로그램, 데이터에 의해 처리된 모든 정보를 말한다.

컴퓨터 소프트웨어는 컴퓨터 하드웨어의 반대 의미로, 컴퓨터 하드웨어는 소프트웨어가 실행되고 저장되는 물리 구조이다. –위키

Utilities for Windows

ColorCop 색상 추출기

  • 설명: 설치 없이 화면에 보이는 색 추출기.
  • 다운로드: http://colorcop.net/download/

μTorrent 유토렌트

  • 설명: P2P 파일 공유, 멀티 다운로드가 가능한 ‘토렌트‘
  • 다운로드: http://www.utorrent.com/intl/ko/desktop/

UmileEncoder 유마일 인코더

  • 설명: 다양한 포멧을 지원하는 동영상 변환 프로그램
  • 다운로드: http://www.umileencoder.com/kr/

Sublime Text 3 서브라임 텍스트

  • 설명: Sublime Text는 그 자체로도 가볍고 쓸만한 텍스트 편집기,
  • 다운로드: http://www.sublimetext.com/3/

Atom 아톰

  • 설명: 깃허브가 개발한 강력한 문서 편집기.
  • 다운로드: https://atom.io/

OneKey Ghost 원키고스트

  • 설명: 드라이버를 쉽게 백업 및 복구해주는 프로그램.
  • 다운로드: http://hangulhwa.tistory.com/42

DaView Indy 다뷰 인디

  • 설명: 오피스, 한글, PDF, CAD 파일까지 모두 호환, 만능 뷰어.
  • 다운로드: http://datools.kr/

K_Subtitle 동영상 자막 맞추기

  • 설명: 자막 안맞을때 한방에 수정하는 프로그램! K_Subtitle.
  • 다운로드: http://kcoder.tistory.com/146

3DP Chip 3DP 칩

  • 설명: ‘3DP 칩’은 CPU, 메인보드, 그래픽등 최적드라이버 연동.
  • 다운로드: http://www.3dpchip.com/3dp/chip_down_kor.php

3DP Net 3DP 넷

  • 설명: 3DP 넷’은 윈도우즈 설치 후 랜카드 드라이버 설치프로그램.
  • 다운로드: http://www.3dpchip.com/3dp/net_down_kor.php
