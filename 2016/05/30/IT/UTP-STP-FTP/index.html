UTP/STP/FTP 케이블

작성일 2016-05-30 Edited on 2017-12-06 In IT 컴퓨터 Disqus:

LAN CABLE 랜케이블

unshielded twisted pair cable의 약어. 이더넷에서 일반적으로 사용되는 케이블. STP 케이블과 비교해서 실드 처리가 되어 있지 않아 잡음에 약하지만, 비용을 낮출 수 있다.

[네이버 지식백과] UTP [unshielded twisted pair cable] (컴퓨터인터넷IT용어대사전, 2011.1.20, 일진사)

UTP, FTP, STP 구조와 설명

UTP (Unshielded Twist Pair : 비차폐연선 )

두 선간의 전자기 유도를 줄이기 위하여 서로 꼬여져 있는 케이블인데 제품 전선과 피복만으로 구성되어 있다. 이건 보통 일반적인 랜케이블에 사용한다.

FTP (Foil Screened Twist Pair Cable)

쉴드 처리는 되어있지 않고, 알루미늄 은박이 4가닥의 선을 감싸고 있는 케이블이다.
UTP에 비해 절연 기능이 좋고 공장 배선용으로 많이 사용한다.

STP (Shielded Twist Pair Cable : 차폐연선)

쉴드라 하는 것은 연선으로 된 케이블 겉에 외부 피복, 또는 차폐재가 추가되는 것을 말한다.
이 것은 외부의 노이즈를 차단하거나 전기적 신호의 간섭을 대폭 줄여준다.


UTP/STP 표준

  • 카테고리 1 - 전화망에 주로 사용. 데이터 전송용으로는 부적합
  • 카테고리 2 - 최대 4Mbps 속도로 전송, 토큰링 네트워크에 자주 쓰였음
  • 카테고리 3 - 최대 16Mbps 속도로 전송, 10 Base T 네트워크에 사용
  • 카테고리 4 - 최대 16Mbps 속도로 전송, 토큰링 네트워크에 사용
  • 카테고리 5 - 최대 100Mbps 속도로 전송, Fast Ethernet용으로 사용, 기가비트 속도 전송 시 8개의 모든 선 사용.
    • Cat. 5는 현재 네트워크 시공이나 전화 시설 등 모든 랜선이 시설되는 장소에서 사용하는 UTP케이블의 표준 규격이라고 할 수 있습니다. 현재 가장 빠른 인터넷도 100Mbps의 광랜이고, 100Mbps 급의 공유기 및 네트워크 장비들도 아직까지 많이 사용하기 때문에 Cat. 5나 Cat. 5e 면 충분히 사용 가능합니다. Cat. 5는 현재 유지 보수용으로만 사용되며, Cat. 5e 규격의 보편화로 현재는 거의 사용되지 않는 규격이라고 보시면 될 것 같습니다.
    • Category 5e Cable (Cat. 5e) 케이블은 최대 1Gbps의 속도로 데이터를 전송할 수 있습니다. 대역폭은 예전에 사용했던 Cat. 5와 동일한 100MHz입니다. Cat. 5e와 Cat. 5의 UTP케이블은 100m 이상 선을 포설할 경우 신호가 현저히 저하가 되기 때문에 끊길 수도 있습니다. Cat. 5e는 현재 사무실이나 가정집에서 실질적으로 많이 사용하고 있으며, 네트워크공사장비 대부분 이 케이블을 사용합니다.
  • 카테고리 6 - 최대 250MHz의 성능과 1Gbps 인터넷 네트워크에 많이 사용됩니다.
    • Cat. 6는 단순 외관으로는 Cat. 5e와 차이가 별로 없습니다. 하지만 피복을 벗기면 4개의 페어와 내부의 십자 모양의 개재로 구성되어 있습니다. 내부 십자형 개재가 각 페어 간의 간섭을 막을 수 있고, 외부 전자파 장애를 최소화할 수 있게 제작되어 있습니다. 아직은 가정용보다는 회사의 서버나 전산망, IT 계열 업체 등에 많이 활용되고 있습니다. Cat. 5케이블 제작보다 약간? 더 어렵다고 생각하시면 됩니다.
    • Category 6e Cable (Cat. 6e) 케이블은 무려.. 최대 10Gbps의 속도로 데이터를 전송할 수 있습니다. 이 속도를 사용하시고 싶으시면 랜카드도 10Gbps를 사용하셔야 합니다. 만약 랜카드가 1Gbps라면, 10Gbps의 속도를 사용할 수가 없습니다. Cat. 6보다 2배 정도 향상된 기능과 케이블 간섭을 최소화한 케이블이라고 보시면 서버용 랜 케이블에 많이 사용합니다. 또한 CAT6e의 경우 수평배선망의 거리에 따른 신호 손실로 55m까지 10GbE를 지원한다고 하네요.
    • 참고 : 현재 쇼핑몰에서 구할 수 있는 CAT6 BOX 케이블의 종류는 제조사를 막론하고 크게 아래 2가지입니다.
      CAT6e - 250Mhz / 24AWG
      CAT6a - 600Mhz / 23AWG
      CAT6e는 250Mhz까지 지원하며 대부분의 인터넷쇼핑몰에서 CAT6라고 표기하고 판매중인 제품의 규격은 CAT6e 입니다.
      CAT6a 케이블은 CAT6e 케이블과 달리 23AWG의 구리심선이 사용됩니다. AWG란 미국 전선규격이고 숫자가 낮을수록 심선의 굵기가 굵은 제품 입니다.
  • UTP 케이블의 심선 두께가 두꺼울수록 전기역학적으로 회선에 흐를 수 있는 전류의 양이 많아지므로 주파수 대역폭도 높으며 10GbE 100m까지 지원 가능합니다.*
    ( LS전선 CAT6a는 IEEE 802.3 10G base-T 지원 표기 / AMP CAT6 U part No. 1427200는 IEEE 802.3an 지원 표기 )
  • 카테고리 7 - 4개의 개별 실드 처리된 STP에 적합하며 최대 600 MHz의 전송을 위해 설계되었다.
    • Cat. 7케이블은 최대 10Gbps의 속도로 데이터를 전송할 수 있습니다. 아직은 Cat. 7케이블의 성능을 발휘할 만한 환경이 구성되어 있는 곳은 많지 않기 때문에 일반 기업 사무실과 가정집의 경우 Cat. 5e와 Cat. 6 UTP케이블만으로도 충분히 고속 인터넷으로 사용 가능합니다. 기존 Cat. 5e와 Cat. 6케이블과 달리 제작이 매우 어렵고, 가격 또한 비쌉니다.
케이블 속도별 구분
10 Base-T
10 : 속도 Base : Baseband 용 케이블 (디지털) T : TP 케이블 / 최대 전송거리

10 Base T : 10Mbps로 통신. 최대 전송거리 100m . UTP3,4,5 사용
10 Base FL : 10Mbps로 통신. FL(Fiber-optic) 광케이블. ST 커넥터 사용
10 Base 2 : 10Mbps로 통신. 최대 전송거리 200m. 일명 Thin 케이블 . BNC 커넥터 사용
10 Base 5 : 10Mbps로 통신. 최대 전송거리 500m. 일명 thick 케이블. AUI 인터페이스(15pin) 사용
100 Base TX : 전송 속도 100Mbps. 최대 전송거리 100m. CAT.5 케이블 사용
100 Base T2 : 전송 속도 100Mbps. 최대 전송거리 100m. CAT.3,4,5 케이블 사용
100 Base T4 : 전송 속도 100Mbps. 최대 전송거리 100m. CAT.3 케이블 사용
100 Base FX : 전송 속도 100Mbps. 최대 전송거리 2km10km. SC 커넥터 사용
1000 Base SX : 기가비트. 전송 속도 1000Mbps. 최대 전송거리 270m550m.
1000 Base T : 기가비트. 전송 속도 1000Mbps. 최대 전송거리 100m. UTP케이블 사용

UTP케이블 연결법 및 구분

  • 다이렉트(Direct) 케이블 : PC-스위치 / 스위치-공유기 연결 시 사용한다.
  • 크로스(Cross) 케이블 : PC-PC / 스위치-스위치 / 공유기-공유기 동종 기종 간 연결 시 사용한다.

RJ 커넥터 RJ connector

배선을 연결하는 데 사용되는 표준 커넥터. RJ-11, RJ-12, RJ-45가 흔히 사용된다. 전화선을 연결하는 데는 일반적으로 RJ-11이 사용되며, RJ-45는 구내 정보 통신망(LAN)에서 비차폐 연선(UTP)을 연결할 때 사용되는 것으로 모듈 단위로 4상 배선을 할 수 있다.
