HTML 특수문자 리스트 엔티티 코드(Entity Code)

HTML이나 XHTML 문서를 코딩할 때 특수 문자(참조 문자)를 써야 하는 경우 종종 있습니다. 이때 항상 인터넷으로 찾다가 그냥 블로그에 저장하자는 생각에 이렇게 저장해 봅니다.

개인적으로 많이 쓰는 것들을 배치한 것입니다.
