Github Pages 란?

Github Pages는 git호스팅 서비스로 유명한 Github에서 제공하는 무료 웹사이트 서비스다.
용량, 트레픽, 광고등의 제약도 없다. 커스텀 URL도 무료로 제공된다. 다만 서버쪽 처리나 데이터베이스등은 제공되지 않는다.

{ :hexo } blog

Hexo는 Node.js 기반 정적 사이트 생성기(Static site generator)의 일종입니다.(다른 생성기가 많이 있더군요.) 이 블로그는 원래 Jekyll로 만들었는데, 제작에 어려움이 있어(배우면 되겠지만) 몇번을 실패 하다가 그나마 쉽게 보이는 Hexo로 옮겼습니다.(다른 이유도 있지만 반응형 테마등등…)

Jekyll 설치 부분에서는 그리 어려움이 없는것 같았습니다. 몇번만 해보면 쉽게(?) 할수 있는것 같지만 그후 플러그인이나 테마 부분에서 쉽지 않더군요. 또한 Hexo는 Node.js로 만든 오픈 소스 프레임워크 이며 라이센스는 MIT License 입니다.

예전의 블로그에서 아쉽다고 생각 했는 부분이 있어서 많은 외부 프로그램들을 넣어서 만들곤 했지만 그래도 아쉽다고 늘 생각을 했습니다. 그러던 중에 우연히 이것을 발견하고 이 프로그램에 매료되기 시작 했네요. 아직 배우기 시작한지 몇일 되지 않아 모든것이 새롭기만 하는 군요.

빠르고 간단하고 강력한 블로그 프레임워크 hexo

홈페이지(Hexo.io)가시면 자세히 나오지만 저 같은 경우에는 몇번인가 실패하고 적용 시키면서 얻은 경험을 적어 보려 하겠습니다. 물론 전 완전 초보 라서 이렇게 경험 몇번 했지만 이글 읽은 분들은 지식이 많은 분들이라 생각해서 그런지 부끄럽네요. 제가 잘못한 부분이 있으면 댓글 부탁 합니다.

완전 초보용으로 써 봅니다. 제가 완전 초보라서…

설치 Installation

먼저 Hexo를 사용하기 위해서는 Node.js와 Git이 필요합니다. windows 사용자라면 또는 완전 처음이라면 해당 소프트웨어를 먼저 설치 해야 합니다.

  • Node.js 한국어사이트 (한국어도 지원되는것 같습니다)
  • Git 다운로드사이트
