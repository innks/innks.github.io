하이퍼 링크?

하이퍼링크(hyperlink, 문화어: 초연결, 하이퍼연결)는 하이퍼텍스트 문서 안에서 직접 모든 형식의 자료를 가리킬 수 있는 참조 고리이다. 이를테면 동영상, 글, 음악, 그림, 프로그램, 파일, 글의 특정 위치 등을 지정할 수 있다. 이는 하이퍼텍스트의 핵심 개념이며, HTML을 비롯한 마크업 언어에서 구현하고 있다. 이 용어는 단순히 링크(link, 고리)라고 줄여 말하기도 한다. 한마디로 누르면 웹 사이트나 프로그램 등으로 이동하는 것이다.

[위키](https://ko.wikipedia.org/wiki/%ED%95%98%EC%9D%B4%ED%8D%BC%EB%A7%81%ED%81%AC)

# 유용한 링크

이번에 쓸 글은 지극히 저의 개인적인 글입니다. 인터넷을 하면서 알게 된 링크들을 모아 저같은 사람과 저처럼 잘 잃어버리는 사람을 위해 이렇게 글을 적습니다. 사실 제가 필요해서 인터넷에 적은 글입니다.

# 유튜브 다운로더

YouTube 비디오 컨버터 - MP3, MP4, AVI로 다운로드
# https://www.onlinevideoconverter.com/ko/video-converter 유튜브다운로드 할때 자주 사용하는 곳입니다.

# 웹 프론트엔드 프레임워크

Bootstrap 4(부트스트랩)
# https://getbootstrap.com/ Bootstrap 4 많이들 사용하는 사이트입니다.
ZURB의 Foundation
# https://foundation.zurb.com/ Foundation은 페이스북, 모질라, 이베이, 야후와 내셔널지오그레픽 과 같은 이름있는 큰 웹사이트에 사용되었습니다.
Semantic UI
# https://semantic-ui.com/ 읽고 이해하기 쉽게 순수한 언어 원리로 조금만 보면 쉽게 할수 있다
YOOtheme의 UIkit
# https://getuikit.com/ 빠르고 강력한 웹 인터페이스를 개발하기 위한 경량 및 모듈 형 프론트엔드 프레임 워크입니다. 솔찍히 2번째로 많이 사용합니다. 기능이 많습니다.
Yahoo의 Pure
# https://purecss.io/ 반응형 CSS 모듈의 작은 세트. 솔직히 사용 하지 않았습니다. 웹상으로는 기본만 있습니다.
MV* framework
# http://todomvc.com/ 사실 잘 모른다 하지만 프레임워크검색에 있어서 올린다.

# HTML / CSS 압축및 압축 해제

CSS Script - JavaScript / HTML5 / CSS3
# https://www.cssscript.com/ css나 기타 소스들을 검색해서 공부할수 있는 사이트 입니다.
(css-unminify) css 압축해제
# https://mrcoles.com/blog/css-unminify/ css소스 압축된것을 해제하는 사이트 입니다.
온라인 JavaScript / CSS / HTML 압축기
# http://refresh-sf.com/ JavaScript / CSS / HTML 압축하는 사이트 입니다.
CSS 2 -> SASS / SCSS 변환기
# http://css2sass.herokuapp.com/ CSS 파일의 내용을 SASS나 SCSS 내용으로 변경하는 사이트 입니다.

# PNG 및 JPEG 압축

TinyPNG
# https://tinypng.com/ JPEG나 PNG파일의 용량을 줄여 줍니다.(대량이미지 변환 가능)
Optimizilla
# http://optimizilla.com/ 이미지 용량 줄이는것 비교할수 있고 대량 이미지 변환 가능.
iLoveIMG
https://www.iloveimg.com/ko/compress-image/ JPG,PNG,GIF 이미지를 자르고 줄이고(사이즈), 압축에 변환까지 할수 있는 사이트
compressor.io
https://compressor.io/compress 이미지 용량을 줄이는 사이트 이지만 한장씩 가능, 하지만 용량은 가장 많이 줄임

# 게임

안드로이드 버그 게임블로그
# http://darkandroid.tistory.com/ 안드로이드 버그 게임을 다운할수 있고 수시로 업데이트 되는 블로그입니다.
안드로이드 버그 게임블로그
# http://goodkandroid.tistory.com 위와 같이 안드로이드 버그 게임을 다운할수 있고 수시로 업데이트 되는 블로그입니다.
고전게임 크래쉬 블로그
# http://jun1517.tistory.com/ 고전 게임 에뮬레이터모음이나 롬파일 다운 가능한 블로그 입니다.
