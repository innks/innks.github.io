{:리눅스} 표준 에러 코드 List of Linux Error Codes

이 블로그를 만지면서 몇번인가 에러가 생겼습니다. 이 에러를 찾다가 생각 해보니 한번 저장하면 다시 고생을 많이 안하겠다 싶어서 이렇게 저장 합니다.
