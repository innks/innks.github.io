마이닝 - nheqminer로 Zcash를 CPU, GPU로 채굴해보기

작성일 2017-07-12 Edited on 2019-02-14 In IT 컴퓨터 Disqus:


지난번에 소개한 가상화폐(암호 화폐) 직접 채굴이라는 포스터를 하였는데 메신저 프로그램으로 했지만 이번에는 사이트의 한 프로그램을 써서 채굴해 보려고 합니다.

제가 선택한 채굴 프로그램은 제목에서도 적었지만 nheqminer라는 프로그램입니다. 이 프로그램은 CPU나 AMD, NVIDIA 모두 채굴 할 수 있는 프로그램으로 저처럼 옛날 컴퓨터로 하려니 부득이하게 결정한 프로그램입니다.

우선 저의 경우는 별 이상이 없이 실행되지만 같은 방법으로 채굴했지만, 컴퓨터의 상태, 그래픽카드의 사양 등 다양한 이유로, 채굴이 성공하기도 하고 실패하기도 합니다. 이때에는 저보다는 땡글 - 비트코인 커뮤니티에 문의해 보시면 도움이 될 것입니다.

본격 채굴해 보기

채굴 풀은 한글 인터페이스가 지원되는 마이닝풀 허브(miningpoolhub.com)를 사용하여 채굴합니다. 가입부터 해 보겠습니다.

이 전 포스트는 가입을 최소로 해서 이야기했지만 이번에는 가입 부분을 상세(?)로 하겠습니다.

홈페이지 왼쪽 위에 회원가입이 있습니다. 클릭하여 가입합니다. 외국이라 가입절차가 비교적 쉽습니다.

한글이니 참 쉽죠. 이제 회원가입을 했으면 로그인을 합니다.
이메일과 비밀번호를 입력하면 됩니다.

Zcash 풀로 이동

로그인하면 해당 풀로 이동하여 설명을 읽어 봅니다.

설명을 읽어 봐도 무슨 말인지는 모르실 것입니다. 그냥 한번 쭉 읽어는 보세요.
그리고 3번 마이너 설정 보시면 됩니다. 위 스샷에는 나오지 않지만 5가지 정도 프로그램이 있습니다.

(1) Claymore miner - Windows, Linux AMD https://bitcointalk.org/index.php?topic=1670733.0 (2) EWBF miner - Windows, Linux NVIDIA https://bitcointalk.org/index.php?topic=1707546.0 **(3) nheqminer - Windows AMD, NVIDIA, CPU** https://github.com/nicehash/nheqminer/releases (4) Genoil's ZECMiner - Windows AMD https://github.com/Genoil/ZECMiner/tree/master/releases (5) Silentarmy miner - Linux AMD, NVIDIA https://forum.z.cash/t/silentarmy-v5-115-sol-s-on-r9-nano-70-sol-s-on-gtx-1070/3580

이것 중에 한 개인 3번을 이번에 해 보도록 하겠습니다. 마이닝 풀 허브에서는 1번을 추천하는데 저의 컴퓨터는 되지가 않고요. 2번은 GPU라 패스 4번과 5번은 보지도 않았습니다.

참고로 1번은 멀티로 동시에 몇 가지를 채굴할 수 있습니다. 2번은 고사향에서 잘된다고 합니다. 4번은 AMD라 패스, 5번은 안되더군요.
이번에는 CPU만으로 채굴을 하려고 하니 3번 밖에 없습니다.

3번 링크를 타고 들어가 봅니다.

nheqminer-0.5c라는 것을 받으려고 합니다.
윈도 사용자시면. zip파일을 적당한 곳에 받아 압축을 해체합니다.
(이때 바이러스감시 중 실시간 감시 부분을 해제하시길 바랍니다.)

압축을 해체하면 위의 스샷처럼 파일이 나옵니다. 이제 배치파일을 하나 만들어 실행만 하면 됩니다.

먼저 공간에 마우스 오른쪽 버튼을 눌러 새로 만들기(W)에서 텍스트 문서를 하나 만듭니다.

참고

이 부분은 텍스트 문서를 배치파일로 변경하기 위해 확장명을 표시하는 부분입니다. 되어 있으면 넘어가셔도 됩니다.

그 후 창의 왼쪽 위에 구성 부분을 클릭하면 메뉴가 나오는데 폴더 및 검색옵션을 클릭합니다.

이번에 폴더옵션 새 창이 나오면 탭 부분에 보기에 들어갑니다.
조금 밑으로 내려가면 알려진 파일형식 파일 확장자 숨기기난의 검사되어 있는 것을 해제하시기 바랍니다.

이제 적용 누르시고 닫으시면 됩니다.

이제 텍스트 문서 -> 배치파일로 만들기 때문에 이름은 start.bat 변경합니다. 그 후 파일 위에서 마우스 오른쪽을 눌러 편집에 들어갑니다.

이제 메모장에 들어왔을 것입니다. 그러면 밑의 글을 복사해서 붙이시고 몇 가지 부분만 수정하시면 됩니다.

CPU만 사용

:: 기본 유저 프로필 **SET USERID=자신의 마이닝풀허브 아이디** **SET WORKERNAME=마이너워크이름** **SET PASSWORD=마이너워크비밀번호** :: CPU 코어 수 SET CPU_THREADS=2 :: 서버 설정 SET LOCATION=asia.equihash-hub.miningpoolhub.com SET PORT=20570 nheqminer -t %CPU_THREADS% -l %LOCATION%:%PORT% -u %USERID%.%WORKERNAME% -p %PASSWORD% pause

위의 것은 쉽게 한다고 풀어 적은 것이고요. 바꿀 것은 userid랑 마이너위크이름, 폐스워드 부분입니다.
마이너위크이름, 페스워드 부분은 자동으로 생성됩니다.

제 것은 밑에 있습니다. 보시면 알겠지만 한 줄로 된 것을 길게 했네요.

제 아이디는 algiddl11이고 워크이름은 office1입니다. 비번은 x
-t의 값인 CPU_THREADS는 자신의 코어수를 적으시면 됩니다.
전 코어가 2개라 2로 적었고 보통은 4입니다. 코어수는 cpu-z를 다운받아 보시면 됩니다.
켜놓고 작업을 할려면 코어수의 1개를 빼시면 됩니다.
즉) 코어가 4개면 2, 또는 3 하시면 됩니다.

그래픽을 사용할것이면 -cd를 넣으시면 됩니다.

GPU & CPU 사용

:: 기본 유저 프로필 **SET USERID=자신의 마이닝풀허브 아이디** **SET WORKERNAME=마이너워크이름** **SET PASSWORD=마이너워크비밀번호** :: CPU 코어 수 SET CPU_THREADS=2 :: 서버 설정 SET LOCATION=asia.equihash-hub.miningpoolhub.com SET PORT=20570 :: 그래픽카드 사용하기 (GPU1개 : 0, GPU4개 : 0 1 2 3) SET GPUDRIVER=0 nheqminer -cd %GPUDRIVER% -t %CPU_THREADS% -l %LOCATION%:%PORT% -u %USERID%.%WORKERNAME% -p %PASSWORD% pause

참고로 -cd 옵션 대신에 -od 옵션을 사용하면 AMD 그래픽 카드도 지원하지만, 테스트 해보진 않았습니다.

1개의 그래픽 카드면 -cd 0, 4개의 그래픽 카드면 -cd 0 1 2 3입니다.

이제 저장하시고 실행을 해 봅니다.

제 컴퓨터가 고물이라 잘 나오지는 않지만, 위의 스샷에서
1.3619 I/s, 1.99746 Sols/s
이것 중에 Sols가 다른 비트코인의 해시라고 보면 됩니다.
즉, 위의 제 컴퓨터는 약 2 Sols 가 나오는 것이네요. 참…….
아! -e는 안 하셔도 됩니다.

이제 마이닝풀허브에 가서 Zcash풀에 마이닝을 확인 하시면 됩니다.

이제 기다리시면 됩니다. 어렵지 않쵸.

참고

- msvcp120.dll 오류가 나는 경우가 있습니다. 이 경우에는 Visual Studio 2013용 Visul C++ 재배포 가능 패키지를 설치하시면 됩니다.
- 혹시 CUDA 드라이브가 필요하시면 이쪽으로 CUDA Toolkit Download 설치 하시기 바랍니다.

클라우드 마이닝

결국에는 이쪽이네요. CPU나 GPU라고 해도 역시 그래픽카드가 있는것이 해시 또는 Sols가 많이 올라갑니다. 부품 사서 설치하고 전기세 및 유지비(건물, 환기 시스템 등) 등을 주면서 고생하느니 전문 업체에 수수료 주고 부탁하는 것이 더 좋겠다는 생각이 듭니다. (사실 장비 구매 비용이 만만치 않아서) 결국 인터넷으로 찾다가 그래도 많이 하고 또한 오래된 사이트(2년 넘게 유지 가능한)를 찾아 쉽고 편하게 하는 것이 좋겠다 싶어서 전 이용하고 있습니다.

이 사이트는 누군가 소개를 하고 소개한 사람과 신규로 오는 분 모두 혜택이 주어지는 것이 있습니다. 프로모션 코드입니다.
신규로 들어와서 결재할 때 3%의 할인을 받고 소개한 분은 자신의 계측에 해당하는 해시파워 업그레이드를 받습니다.

즉, Win Win 시스템이죠 새로이 하시는 분도 이렇게 소개를 하시면 됩니다.

프로모션 코드: **`QOWoZm`** (3% 할인, 재사용 가능) 대소문자 구분합니다.

위 그림을 클릭하면 제네시스 마이닝 사이트로 이동합니다.
  1. 제네시스 마이닝의 마케팅 방법중 하나는 이렇게 정보를 남기고 추천하는 시스템입니다.
  2. 프로모션 코드를 입력하면 결제금액의 3%가 할인받을 수 있습니다.
    (입력한 사람은 구입할 때에 3% discount 할인을 받고 코드를 제공자는 계측에 해당하는 해시 파워 업그레이드를 받습니다.)
  3. 신용카드 결제 시 결제금액이 100달러가 넘지 않으면 5불의 추가요금이 발생합니다.
  4. 신용카드 결제 시 30일 동안은 출금이 제한됩니다. (카드사 승인 기간)
